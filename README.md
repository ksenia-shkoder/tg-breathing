# Breathe Flow Mini App

Готовый статический прототип Telegram Mini App для дыхательных практик.

## Что уже сделано

- многоэкранный интерфейс: `Сегодня`, `Программы`, `Прогресс`, `Профиль`
- дыхательная сессия с таймером и анимацией
- `check-in` до и после практики
- сохранение сессий и состояния в `localStorage`
- вычисляемый `streak`, недельная цель, heatmap и рекомендации
- интеграция с `Telegram WebApp API`
- кнопка отправки прогресса через `Telegram.WebApp.sendData`

## Файлы

- `index.html` — интерфейс Mini App
- `styles.css` — визуальный стиль
- `app.js` — логика приложения и Telegram WebApp
- `vercel.json` — конфигурация для статического деплоя

## Как открыть локально

Можно открыть [`index.html`](./index.html) прямо в браузере.

Если нужен локальный сервер:

```bash
python3 -m http.server 4173
```

Потом открыть `http://localhost:4173`.

## Как получить публичную ссылку

Самый быстрый путь — Vercel или Netlify.

### Вариант с Vercel

1. Создай новый репозиторий и отправь туда эти файлы.
2. Импортируй репозиторий в Vercel.
3. В настройках проекта используй тип `Other`.
4. После деплоя получишь HTTPS-ссылку вида `https://your-project.vercel.app`.

## Как подключить в Telegram

1. Создай бота через BotFather.
2. Открой BotFather и настрой Mini App URL через команду `setmenubutton` или `newapp`.
3. Укажи публичную HTTPS-ссылку на задеплоенное приложение.
4. Открой бота в Telegram и запусти приложение через меню.
